# ClaudeCode1

A Python 3.11 project containing experiments built with Claude Code.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
```

## Projects

### findme.jobs landing page (`findme_jobs/`)

A Flask landing page for findme.jobs — an AI-powered interview prep and job matching product.

```bash
cd findme_jobs
pip install -r requirements.txt
python app.py
```

Opens at **http://localhost:8080** (port 8080 — macOS AirPlay uses 5000).
